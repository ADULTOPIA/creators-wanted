# 2026 Adultopia 大人國 - Creators Wanted 公式サイト制作仕様書

## 1. プロジェクト概要
本プロジェクトは、2026年開催のイベント「Adultopia 大人國」におけるクリエイター出展募集専用のランディングページ（LP）制作です。Reactを用いたモダンなフロントエンド構成で、GitHub Pagesによる静的ホスティングを前提としています。

- **サイト名**: Creators Wanted 公式サイト
- **開発用URL**: [http://localhost:3000/creators-wanted](http://localhost:3000/creators-wanted)
- **ターゲット**: クリエイター、コスプレイヤー、KOL、サークル出展者

---

## 2. 技術スタック・開発環境

### コア技術
- **Framework**: React (Create React App)
- **Language**: TypeScript
- **Styling**: styled-components
- **Routing**: React Router
- **SEO/Meta**: React Helmet
- **Hosting**: GitHub Pages

### 開発コマンド
- **Node.js**: v14以上 / **npm**: v6以上
- **起動**: `npm start`
- **ビルド**: `npm run build`
- **デプロイ**: `npm run deploy` (または `./deploy.sh`)
