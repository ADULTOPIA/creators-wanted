# Creators Wanted 公式サイト

2026 Adultopia 大人國 - 創作者招募ウェブサイトプロジェクトです。

## サイトURL

公式サイトは以下のURLでアクセスできます：
[https://adultopia.github.io/creators-wanted/](https://adultopia.github.io/creators-wanted/)

## プロジェクト構成

- `Doc/` - 仕様書やリソース
- `creators-wanted/` - React アプリケーション

## 開発方法

React アプリケーションのディレクトリに移動し、開発を行います。

```bash
cd creators-wanted
npm install
npm start
```

詳細は [creators-wanted/README.md](./creators-wanted/README.md) を参照してください。
